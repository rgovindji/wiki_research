# Feedback Log — Framework Calls & Outcomes

## What this is

A publicly maintained accountability log of the wiki's calls and how they played out. **The wiki's claims aren't worth anything unless we check them against reality.** This page tracks:

1. **Calls the wiki made** (stance changes, conviction upgrades/downgrades, framework predictions, scenario probabilities)
2. **What actually happened** (with dates)
3. **Honest verdict** (right, right-but-timing-wrong, partially right, wrong, process gap)
4. **Lesson captured** (so the framework improves)

This is the wiki's "show your work" artifact. **Readers can decide whether the analytical framework is worth trusting based on the actual hit rate**, not on glossy after-the-fact selection.

## Schema

Each entry follows this structure:

```
## [VERIFICATION DATE] {Verdict tag} | {Short title}

**Original call (date):** What the wiki said and why
**Observed (date):** What actually happened
**Verdict:** [Right / Right but timing wrong / Partially right / Wrong / Process gap]
**Framework:** Which wiki framework or page was used
**Lesson:** What we'd do differently or what to retain
```

## Verdict legend

- **✅ Right** — Outcome matched the call directionally AND within the rough timeframe
- **⏱️ Right but timing wrong** — Call was structurally correct but the timing was off (stock moved opposite direction in the short term before the structural call played out)
- **〰️ Partially right** — Call was directionally correct on some dimensions, wrong on others
- **❌ Wrong** — Call was directionally incorrect
- **🛠️ Process gap** — The wiki missed capturing something it should have; not necessarily wrong analytically, but the workflow broke

## Lessons accumulated

1. **Stance ≠ market timing.** Conviction levels signal position-sizing discipline. Downgrade conviction = lower bet + more exit triggers, NOT predict near-term decline.
2. **Indicator firing ≠ stock down tomorrow.** Late-cycle warning indicators describe asymmetric risk, not catalysts.
3. **Cross-source convergence is the strongest signal.** When two independent sources reach the same answer within hours, treat as a high-confidence anchor.
4. **Macro frameworks need idiosyncratic-catalyst overlay.** Before publishing macro positioning, check the next 7 days of earnings catalysts for every name in the cohort.
5. **Fact-based downgrades are more durable than forecast-based downgrades.** Competitive position + financial profile metrics are checkable; forecasts are not.
