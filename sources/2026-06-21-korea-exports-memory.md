# South Korea preliminary exports, June 1-20 2026 — memory supercycle, real-time

## TL;DR
- **The memory supercycle is broadening from DRAM into NAND/SSD/HBM, and HBM is re-accelerating.** Read the **MoM** column (YoY is base-effect-distorted): DRAM is plateauing (+3% MoM) while NAND (+28%), SSD (+25%) and HBM (+51%) accelerate sequentially.
- Korea customs first-20-days = the cleanest near-real-time read on memory demand (SK Hynix + Samsung are Korean) — harder data than contract-price estimates.

## The data (June 1-20 2026, YoY / MoM)
| Category | YoY | MoM |
|---|---|---|
| DRAM (incl. modules) | **+342%** | **+3%** |
| Flash / NAND | +336% | **+28%** |
| SSD | **+405%** | +25% |
| MCP (HBM) | +209% | **+51%** |

## Key reads (with confidence)
- **HBM (MCP) +51% MoM = the standout.** Sequential re-acceleration into the HBM4 / Vera-Rubin ramp. Lowest YoY (+209%) only because HBM was already ramping a year ago (small base). *High-signal bull for [[MU]] / SK Hynix / Samsung HBM.*
- **NAND +28% MoM / SSD +25% MoM = hard macro confirmation of "NAND is catching DRAM"** (Dylan Patel, [[2026-06-20-dylan-patel-daytona-cpu-bottleneck]] + the SemiAnalysis DeepSeek/NAND read). Enterprise SSD +405% YoY = the AI-data-exhaust storage thesis printing. *Bullish [[MU]] NAND leg + SNDK/WDC.*
- **DRAM +3% MoM = the one yellow flag.** After DRAM contract ASPs ran ~+90% in Q1, sequential DRAM is *plateauing*. Not a rollover, but the first hint the DRAM ASP curve is flattening while NAND/HBM take the baton — the data point the bears ([[ai-bubble-debate]] Future Horizons "ASP-driven not unit-driven"; Jonathan Ross "memory gets solved again") will cite. *Watch for the first negative m/m DRAM print as the falsifier.*

## How this is used (and the bias note)
- **Demand confirmation, not froth proof.** A near-real-time export surge cuts against the "fabricated/over-extrapolated demand" bear — but it's *ASP-heavy* (the YoY is price, not unit), so it simultaneously feeds the "narrow, price-led cycle" worry. Surface both.
- Cyclical-peak discipline still applies: this confirms the *cycle is real and broadening*, it does NOT change the entry discipline on memory equities already at/near all-time highs ([[MU]], SNDK) — own the supercycle's *rent*, not its *whiplash*.
